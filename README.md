# Jointt Website

A React recreation of the Jointt website originally built on Framer.

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Features

- Hero section with call-to-action
- Statistics section
- Benefits showcase
- Advanced capabilities
- Features grid
- Pricing information
- FAQ section with accordion
- Responsive footer
- Smooth scrolling navigation

## Tech Stack

- React 18
- Vite
- CSS3
